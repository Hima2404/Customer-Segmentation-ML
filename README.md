->Pandas
->Matplotlib
->Scikit-Learn
*Dataset
->Mall Customers Dataset(From Kaggle)
*Features
->Data Analysis
->Elbow Method
->Customer Segmentation
->Customer Type Classification
*Output
->Elbow Method Graph
->Customer Segmentation Output CSV
